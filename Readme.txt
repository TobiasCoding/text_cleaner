Its a simply program in which you'll be able to clean and separate words united (or not), as 'holacomoestas?', in a text string. It use a spanish dictionary to parse the input text. Also it will be possible add an another languaje dictionary. The spanish dictionary was getted from https://olea.org/proyectos/lemarios/ and then were manually added some terms as vocals, numbers and non-alphanumeric symbols.
